**Productos cerrados:**

1. **Chat IA / Chatbot** — agentes conversacionales sobre WhatsApp, Web, Instagram.
   - IA real: LLM (Claude Haiku/Sonnet, GPT-4o-mini) con RAG sobre docs del cliente.
   - "Falsa IA" / decision tree: árbol + templates, para flujos transaccionales (agendar, FAQ, delivery).
   - Híbrido (recomendado): árbol primero, fallback a LLM cuando sale del flujo conocido.

2. **CRM** — setup, migración, automatización + IA sobre HubSpot, Pipedrive o Salesforce.
   - La licencia la paga el cliente directo (queda a su nombre, sin lock-in).
   - Capa IA: lead scoring, email drafts, resúmenes de reuniones, forecast.

3. **Software / Web** — WordPress, Astro/Next/Node, MVPs, integraciones, SEO, redes sociales.
   - Sub-tabs en la landing: Web (WP+Code), Software a la medida, SEO, Redes Sociales.

**Producto eliminado:** ERP (ver `feedback_no_erp.md`).

**Orden de prioridad para construir** (decidido 2026-05-17):
1. Chat IA híbrido sobre WhatsApp → primer producto vendible, demo visual fuerte.
2. CRM "engagement" base (companies + contacts + deals + notes).
3. Capa IA sobre CRM (scoring, drafts, resúmenes).
4. Redes sociales (más complejo: integraciones Meta Graph, LinkedIn, TikTok API).

**Target:** 30-50 clientes para validar el modelo. Hoy en validación, sin pipeline grande.
